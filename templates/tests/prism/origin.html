1. 在 settings 的 CKEDITOR_CONFIGS 里的 extraPlugins 对应的 value 里加入插件 'prism' 和 另外两个插件："lineutils"、"widget"（这两个插件无须下载，在 django-ckeditor 中已经有了）
# ckeditor configs

CKEDITOR_CONFIGS = {

    'language': 'zh-cn',

    'awesome_ckeditor': {

        'toolbar': 'full',

        'width': 'auto',

        'height': '480px',

        'tabSpaces': 4,

        # 插件

        # prism：代码高亮、代码行数

        # uploadimage：允许用户直接在编辑器里粘贴（ctrl + v）图片

        # codesnippet：添加代码块的功能

        'extraPlugins': ','.join(['codesnippet', 'uploadimage', 'prism', 'widget', 'lineutils', ]),

    },

}
